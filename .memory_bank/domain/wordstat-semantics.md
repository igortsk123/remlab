## Вердикт по неявному спросу (гипотеза владельца «дешевле, строим воронку», 2026-07-11)

**Берём (наш интент внутри неявного):** «сколько стоит ремонт квартиры» 52k + «ремонт в
новостройке» 22k + консьюмерская «смета» ~6k + «с чего начать ремонт» 16k ≈ **~95k/мес
mid-funnel** — лендинг «прикинь бюджет и посмотри результат до бригады» → ступень 3 (Cost
Engine/смета). Это эшелон 3 стратегии.

**Не берём в лоб на Поиске (возражение, аргументы):**
- «ремонт под ключ» (155k): интент = найм бригады, не инструмент; аукцион подрядчиков с чеком
  0.5–2 млн ₽ — CPC сотни ₽ (не «дешевле»), сейчас пик сезона. Нерелевантный оффер → низкий CTR →
  ещё дороже клик + низкая конверсия лендинга. Опциональный микротест позже: «ремонт комнаты под
  ключ» 10k с оффером «сначала посмотрите результат», жёсткий лимит.
- мебель «недорого» (~150k): marketplace-интент («авито», «б/у», «с доставкой»), спрос −50% за
  2 года; free-affiliate экономика на холодном платном клике не сходится: ~3% с выкупа ≈
  600–1 800 ₽ дохода, сквозная конверсия «клик→фото→реф-клик→выкуп» — доли % → CAC ≫ доход
  (kill-критерий мастер-плана). Мебельный неявный спрос → **РСЯ** по интересам с крео
  «примерь мебель в свою комнату по фото» (там CPC действительно низкий) — после валидации Поиска.

Wordstat не даёт CPC — перед запуском сверить «Прогнозом бюджета» в UI Директа (допущение о
дорогом «под ключ» проверить там же).

## Сезонность (dynamics, 24 мес: 2024-07 → 2026-06)

- **«ремонт квартиры»:** ярко выражен пик **июль–ноябрь** (авг-2025 923k, окт-2025 896k) и дно
  дек–фев (~590–617k). Сезон стартует В ИЮЛЕ — сейчас (июль 2026) вход в горячий сезон.
  YoY рост: лето-2025 к лету-2024 ≈ +60–90%.
- **«дизайн интерьера»:** пик **октябрь–январь** (ноя-2024 267k, янв-2025 289k) — «планируют зимой,
  ремонтируют летом». Июнь-2026 аномально высок (264k).
- **«нейросеть дизайн интерьера»:** июл-2024 1 427 → пик ноя-2025 4 801 → плато ~3 000–3 200
  с зимы 2026. Категория выросла ×2 за 2 года, momentum замедлился — рынок ещё НЕ мейнстрим,
  окно раннего входа открыто.
- **«дизайн комнаты» / «ремонт комнаты»:** ровнее; локальные пики окт–ноя и январь.
- **«ремонт под ключ»:** пики июнь–сентябрь (июнь-2026 173k — максимум за 24 мес, YoY +45%),
  дно дек–фев (~80k) — аукцион подрядчиков сейчас на пике.
- **«недорогая мебель»:** монотонное падение 105k (июл-2024) → 43–55k (2026), −50% за 2 года.

## Импликации для плана рекламы (сам план — отдельно, по agent-workflow)

1. **Эшелон 1 (старт, Поиск):** кластер A + «онлайн/бесплатно»-часть кластера B — узко, дёшево,
   intent=продукт. Оффер: «бесплатно, по фото, на русском, за минуты» → free-ступень.
2. **Эшелон 2:** бренды конкурентов (F: Remplanner/Planner 5D/Планоплан — суммарно ~60k) —
   в ключах можно, в текстах чужие бренды НЕ использовать (модерация).
3. **Эшелон 3 (paid-оффер + mid-funnel цены):** «дизайн проект квартиры/комнаты + стоимость»
   (48k) и чистые ценовые срезы ремонта — «сколько стоит ремонт квартиры» 52k, «ремонт в
   новостройке» 22k, «смета» ~6k, «с чего начать» 16k (итого ~95k) → лендинг «прикинь бюджет и
   посмотри результат до бригады», room pack «в 10–30× дешевле дизайнера», ступень 3 Cost Engine.
4. **РСЯ и широкие маски (C/D/G)** — только после валидации конверсии Поиска (правило из
  health-card); картинки до/после — наш формат; мебельный неявный спрос (G) — крео
  «примерь мебель в свою комнату по фото». «Ремонт под ключ» на Поиск в лоб НЕ брать (§Вердикт).
5. **Тайминг:** сезон ремонта уже идёт (июль–ноябрь), дизайн-пик — окт–янв → запуск Поиска
   не откладывать; к октябрю расширяться на дизайн-маски.
6. **Пререквизиты:** счётчик Метрики для remont-lab.online (НЕТ) + цели воронки; кампании
   создавать в общем Директ-аккаунте отдельно от health-card (их `708745261` не трогать);
   autotargeting сразу на 0.30 ₽ (грабли соседей: слив 5k ₽/час).

## CPC-прогноз Директа (снято 2026-07-11 по API: keywordbids + CreateNewForecast, гео 225)

По всем 69 активным ключам Этапов 1–2 (кампании 712722343/712722345, точная фраза):
- **Вход в показы** (объём ~5): ремонт 6–18 ₽, калькуляторы 11–20 ₽, хвосты «сколько нужно…» 25–38 ₽.
- **Заметный объём** (~75): ремонт 30–95 ₽, калькуляторы 40–97 ₽ — везде ВЫШЕ потолка BidCeiling 20 ₽
  → с текущим потолком показы будут внизу и частично; «+48ч нет показов → поднять потолок» из
  протокола мониторинга — ожидаемый сценарий, не аномалия.
- **Показы точной фразы** (прогноз Директа/мес): у большинства ключей десятки-сотни (кавычки режут
  широкий спрос из оценок `[~N тыс/мес]` в разы); звезда — «сколько стоит ремонт квартиры»
  **35 893**/мес точной фразой (больше всех остальных ключей вместе). Топ калькуляторов:
  «калькулятор обоев» 3 872, «калькулятор плитки» 1 002, «калькулятор ламината» 956.
- Гоча API: Wordstat search-api квота **100 запросов/час**; фразы в кавычках topRequests не
  принимает (пустой ответ). v4 `CreateNewForecast` жив, требует body в UTF-8 без \u-эскейпов
