# Стоимость работ РФ — медианы Москвы + регионы (снимок 2026-07-11)

> Питает вход Б «сколько стоит ремонт» (`lib/pricing/works.ts`). Медианы масс-рынка (наш юзер —
> эконом/DIY), НЕ премиум-компании. Веб-ресёрч 2026-07-11. План — `plans/pricing-db-ru.md`.

## Медианы работ Москвы (₽/м², масс-рынок)
| Работа | ₽/м² (Мск) | Диапазон источников |
|---|---|---|
| Грунтовка | 100 | profi низ ~90 |
| Покраска стен | 300 | profi 90 … titan 800 |
| Поклейка обоев | 400 | profi 300 … titan 800 |
| Шпаклёвка стен | 550 | profi 400 … titan 1800 (премиум) |
| Стяжка пола | 850 | profi 850 … titan 1400 |
| Укладка ламината | 450 | ~400–600 |
| Укладка плитки | 1200 | profi 600 … titan 2500 |
| Демонтаж | 300 | ~200–400 |

Материалы (₽/м², федеральные — коэф. региона НЕ применяется): грунтовка 40, шпаклёвка 130,
краска 180, обои 350, ламинат 800.

## Региональные коэффициенты к Москве (применяются к РАБОТАМ)
Обзоры (banki.ru, cian, 101-app): регионы дешевле Москвы на 20–30% в работах; коэф. 0.6–0.8.
Материалы почти не меняются (маркетплейсы). Сетка групп (`REGION_K` в коде):
| Группа | k | Примеры |
|---|---|---|
| Москва | 1.00 | — |
| Санкт-Петербург | 0.88 | — |
| Город-миллионник | 0.70 | Екб, Новосиб, Казань, Краснодар, Н.Новгород… |
| Средний город (100–500 тыс.) | 0.62 | — |
| Малый город / посёлок | 0.55 | — |
| Дальний Восток / Север | 0.95 | Сахалин, Камчатка, Якутия (логистика) |

Юзер уточняет регион селектором на `/calc/remont` (дефолт Москва). GeoIP по IP — Фаза 2.
