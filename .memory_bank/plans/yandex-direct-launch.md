## Цель
Запустить первую поисковую кампанию remlab в Яндекс Директе на горячем AI-кластере (~15–18k
показов/мес) с защитой от слива по всем граблям v0-health-card, с работающей Метрикой и
воронкой целей. Заливка через Direct API (доступ есть, `_secrets/ACCESS.md`).

## Источник задачи
Владелец (2026-07-11): «пиши план на рекламные кампании, учти опыт v0-health-card, спорное
гугли; по итогу подтверждённого плана создай группу объявлений и залей всё в Директ; картинки —
потом отдельным планом». Семантика — `domain/wordstat-semantics.md`; стратегия —
`core/marketing-acquisition.md`.

## Уроки health-card, зашитые в план (источник: их `advertising/*`)
1. **Автотаргетинг = главный слив** (2 инцидента: −5 519 ₽/час и −1 110 ₽/полдня). Протокол:
   после `adgroups.add` НЕМЕДЛЕННО `bids.set` 0.30 ₽ на AT-ключ (`id = 200000000000+group_id`)
   и `keywords.update` категорий → только `EXACT` (осталные NO) — и лишь потом keywords/ads.
2. **Стратегия**: их выстраданная связка — `WB_MAXIMUM_CLICKS` + `WeeklySpendLimit` +
   `BidCeiling` (реальный CPC длиннохвоста 12–18 ₽ при потолке 18 ₽), РСЯ `SERVING_OFF`.
   Ручная HIGHEST_POSITION жгла дневной бюджет за ~30 мин (STANDARD-режим перебирает до 130%).
3. **Settings**: `CAMPAIGN_EXACT_PHRASE_MATCHING_ENABLED: YES`, `ALTERNATIVE_TEXTS_ENABLED: NO`,
   `ENABLE_AREA_OF_INTEREST_TARGETING: NO`.
4. **Фразовое соответствие** ("кавычки") на всех ключах + минусовка по отчёту «Поисковые
   запросы» с первых часов. Минимум 2 объявления на группу.
5. **Метрика**: SPA Next.js не шлёт pageview сам — нужен `ym('hit')` на смену роута; цели-action
   считаются только с даты создания (не удалять, архивировать); UTM с `{source_type}`.
6. **После создания групп** — верификация через `bids.get`/`keywords.get` (не верить «принято»).

## Проверено в интернете (спорные точки)
- **ЕПК заменила ТГО в интерфейсе** (с 09.2024), но API v5 `campaigns.add` по-прежнему
  документирует `TextCampaign` → план: TEXT_CAMPAIGN через API (как живая кампания соседей);
  если add отклонит тип — fallback `UnifiedCampaign` (только Поиск). Источники:
  yandex.ru/dev/direct/doc/ref-v5/campaigns/add, sv-digital, elama.
- **CPC ниши**: «ремонт под ключ» Мск ~194 ₽ / регионы ~62 ₽ (demis, mst) — подтверждает «в лоб
  не брать»; инструментальные длиннохвосты ≈ здоровый диапазон 10–30 ₽ → BidCeiling 20 ₽ старт.
- **Брендовые ключи конкурентов**: Директ не запрещает, но юр. риск (ФАС ст.14.33 — 100–500 тыс.,
  суд по ТЗ — до 5 млн; onlinepatent, 1ps, yagla). Чужой бренд в ТЕКСТЕ объявления — нельзя
  точно. Легальная альтернатива — категория автотаргетинга `COMPETITOR`. → решение владельца ⏸.

## Скоуп — что входит (Фаза 1)
- Метрика: счётчик + 6 целей воронки + код на сайт (layout, SPA-hit, trackGoal) + деплой.
- Футер сайта: реквизиты продавца (требование модерации) — текст подтверждает владелец ⏸.
- Одна кампания Поиск (TEXT_CAMPAIGN) + 4 группы (см. ниже) + 8 объявлений + sitelinks.
- Минус-слова кампании (стартовый список), защита от слива по протоколу.
- Мониторинг-гейты: +2ч, +24ч, +48ч, +7дн.

## Скоуп — что НЕ входит
- РСЯ и картинки (Фаза 2 — отдельный план: промпты для генерации/где искать изображения).
- Mid-funnel «сколько стоит ремонт» (эшелон 3) и широкие маски — после валидации Фазы 1.
- Брендовая группа конкурентов — только после решения владельца (Фаза 1b).
- Авто-минусовка (Remote Routine, как у соседей) — follow-up после запуска.
- Платёжная воронка/цены на сайте не трогаем.

## Порядок заливки (Direct API; строго по шагам)
1. `campaigns.add` (TEXT_CAMPAIGN, параметры выше, NegativeKeywords сразу) → campaign_id.
   Верификация `campaigns.get` (стратегия/лимиты/Network=SERVING_OFF/Settings).
2. `adgroups.add` (4 группы) → group_ids.
3. **СРАЗУ** для каждой группы: `bids.set` AT-ключа (200000000000+gid) = 300000 (0.30 ₽) и
   `keywords.update` AutotargetingCategories: EXACT=YES, остальные NO. Верификация `bids.get`.
4. `keywords.add` (фразы по группам, кавычки). 5. `sitelinks.add` → set_id.
6. `ads.add` (2×4 объявления, VCard НЕТ на старте, sitelinks set) → `ads.moderate`.
7. Верификация: `ads.get`/`keywords.get` статусы; кампания остаётся с State=OFF/без денег до ⏸.
8. Смоук лендинга: 200 по URL с UTM (`curl -I 'https://remont-lab.online/?utm_source=yandex…'`).

## Запуск и мониторинг (после модерации + пополнения баланса ⏸)
- **+2 ч**: отчёт «Поисковые запросы» → минусовка мусора; проверить расход/CPC (потолок 20 ₽).
- **+24 ч**: пейсинг недельного лимита (≈500 ₽/день), CTR по группам (<3% на Поиске — переписать
  тексты), доля автотаргетинга в расходе (>20% — проверить категории).
- **+48 ч**: если показов почти нет — поднять BidCeiling до 25 ₽ (шаг 5 ₽, не выше 30 без ⏸).
- **+7 дн**: воронка Метрики (визит→фото→превью), CPC факт, решение по Фазе 1b (бренды) и
  эшелону 3; отчёт владельцу. Kill-гейт: расход ≥ 2×лимита или CR в фото <1% при 300+ кликах →
  пауза кампании + разбор.
