## Цель
4 UI-правки на /calc/plitka: подсказку «Введите размеры…» разместить НАД кнопками добавления
(и для стен, и для пола), показать ненавязчивый значок, что имя комнаты редактируемо, и оформить
призыв «задайте размер плитки…» курсивом в скобках без тире.

## Источник задачи
Промпт владельца по /calc/plitka:
1. «Стены / Введите размеры…» — над «+ добавить размеры стены».
2. «Пол / Введите размеры…» — над «+ добавить размеры пола» (сейчас у пола без блока подсказки нет).
3. Показать, что названия комнат редактируются — неброский значок (карандаш).
4. «задайте размер плитки или вставьте ссылку» — в скобках, курсивом, без «—» перед ним.

## Скоуп — что входит
- **Порядок стен:** note (пустая подсказка/результат) переместить ВЫШЕ кнопки «+ добавить размеры
  стены» (сейчас ниже). Плитка + прочие стеновые виды.
- **Пустой блок пола:** когда пол не добавлен — показать note «Пол / Введите размеры…» НАД кнопкой
  «+ добавить размеры пола». Вынести общий текст подсказки в константу; helper `EmptyNote`.
- **Имя комнаты:** рядом с полем имени — приглушённый значок-карандаш (SVG, muted), сигнал «можно
  редактировать». Клик по значку — фокус в поле (не обязательно, но желательно).
- **qtyUnknown-текст:** `? шт (задайте размер плитки или вставьте ссылку)` — «? шт» акцентом,
  скобочная часть курсивом (em, обычный вес), тире «—» убрать.

## Скоуп — что НЕ входит
- Логика расчётов/парсинга/контракты — не трогаем.
- Другие виды калькулятора по существу (только общий порядок note/кнопки, т.к. код общий).
