## Цель
Продумать порядок экранов и тексты (микрокопирайт) для нового двухэтапного флоу, где выбор пользователя
управляет генерацией. Подплан к [[interactive-object-selection]]. Дизайн/тон — за владельцем; здесь черновик
для правок.

## Порядок экранов (остаётся 5; стиль сливается в экран выбора)
| # | Маршрут | Экран | Изменение |
|---|---------|-------|-----------|
| 1 | `/start` | Цель: что сделать с комнатой | без изменений |
| 2 | `/p/[id]/brief` | Фото + бриф (город/бюджет/ограничения) | **убрать «Что оставить»**; **стиль тоже убрать отсюда** |
| 3 | `/p/[id]/select` **(НОВЫЙ, заменяет `/style`)** | «Что меняем на фото?» — на монтировании идёт анализ («Анализируем фото…»), затем: объекты (кнопки) + **выбор стиля** + одно поле-пожелание + миниатюра фото → «Сгенерировать» | объединяет разбор объектов и выбор стиля |
| 4 | `/p/[id]/preview` | Результат («магия происходит» пока генерит — как сейчас) | генерация запускается по кнопке с экрана 3 |
| 5 | `/p/[id]/paywall` | Полный план | без изменений |

`Progress total = 5`. Старый отдельный экран `/style` убираем (стиль переехал на экран 3).
Порядок (решение владельца): сначала фото → **анализ** → потом единый выбор (что оставить/убрать/поменять +
стиль + пожелание) → генерация.

## Экран 4 — «Что меняем на фото?» (новый) — тексты (черновик)
- **Заголовок:** «Что меняем на фото?»
- **Подзаголовок:** «Мы разобрали вашу комнату. По каждому предмету выберите: оставить, поменять или убрать.
  А ниже одним полем опишите словами, каким хотите видеть результат.»
- **Состояние «разбираем»** (пока идёт analyze): «Разбираем фото…» + спиннер.
- **Список объектов** — строка = `{название}` + три кнопки: **Оставить · Поменять · Убрать**
  (по умолчанию — предложение ИИ; пользователь может переключить).
- **Fallback** (если объекты не распознались): общий список «Стены · Пол · Свет · Текстиль и декор» с теми же
  кнопками + подпись «Не удалось точно распознать предметы — отметьте по общим зонам или опишите пожелание ниже».
- **Поле-пожелание (одно, на весь кадр, необязательное):**
  - label: «Ваши пожелания»
  - placeholder: «Опишите словами, что хотите: цвет стен, какую мебель поставить, общее настроение… Чем конкретнее — тем точнее результат.»
- **Выбор стиля:** карточки стилей (перенесены со старого экрана `/style`) — тут же на экране.
- **Миниатюра исходного фото** — показываем (владелец: «оставь, посмотрим»).
- **Кнопка:** «Сгенерировать комнату»
- **Подпись под кнопкой:** «Это визуальная концепция комнаты, а не рабочий проект. Размеры проверяйте перед покупкой.»
- **Экран 4 (генерация) — загрузка:** оставить как сейчас («магия происходит»-стиль).
