# Доводка ключей и минус-фраз (Этапы 1–4)

## Цель
По каждой кампании: полный продуманный список минус-фраз (по реальным хвостам Вордстата,
не по догадкам) + добавка недостающих ключей. Применение через API — после «деплой».

## Источник задачи
Владелец 2026-07-11: «продумай ключевые слова все и минус-фразы по каждой кампании отдельно».

## Метод (что сделано)
Все 80 действующих ключей прогнаны через Wordstat topRequests (до 150 вложенных фраз на ключ,
РФ); из хвостов частотным анализом добыты «лишние слова» с весом = сумма показов фраз. Мусор →
минус; полезное → кандидат в ключи. Минус-слова в Директе лемматизируются (одна форма кроет
словоформы). Сырьё: scratchpad `ws_mine/*.json` (эфемерно; квота Вордстат на сегодня выбрана).

## Этап 1 — AI-дизайн (712721026, ACCEPTED; 66 минусов уже стоят)

**Минусы ДОБАВИТЬ (веса из хвостов):**
| Слово | Вес | Почему (пример) |
|---|---|---|
| 3d, 3д | 14k+4.5k | «дизайн интерьера 3d онлайн» — аудитория 3D-планировщиков (рисуют сами), не фото-подход |
| проект | 6.4k | «дизайн проект комнаты онлайн» — оффер эшелона 3 (room pack); ⚠️ СНЯТЬ минус при запуске эшелона 3 |
| промт, промпт | 0.2k | «промт для визуализации интерьера» — ищут промпты для чужих нейросетей |
| начинающих | 0.9k | «программы для дизайна интерьера для начинающих» — обучение |
| москва, спб | 0.6k | «визуализация интерьера москва» — заказ услуги у студии |

**НЕ минусовать (осознанно, наша ЦА):** самостоятельно/самому/самой (5.7k+3.1k — сам-онлайн
= мы), мебелью/расстановки (ключи группы), конструктор, приложение, без регистрации, детской
(fake-door сигнал), дизайнеров (будущий B2B), русские, программы (ключ группы).

**Ключи ДОБАВИТЬ:** `"нейросеть для ремонта"` (1 080, чистый) — группа «Нейросеть/ИИ дизайн»;
`"дизайн комнаты с мебелью онлайн"` (вес 2.1k) — группа «Дизайн онлайн».

## Этап 2 — Бюджетный ремонт комнат (712722343, SUSPENDED; 55 минусов стоят)

**Главный найденный мусор — НЕДВИЖИМОСТЬ:** «купить квартиру недорого без ремонта», «квартиры
в хрущевках без ремонта», «сколько стоят квартиры…» (вес «купить» 8k, «вторичка» 2.9k).
**Минусы ДОБАВИТЬ:**
| Слово | Почему |
|---|---|
| вторичка | покупка жилья |
| продажа, продам | продажа жилья |
| аренда, сниму, снять, посуточно | аренда жилья |
| ипотека | сделки с жильём |
| ванна | усиление семейства (стоят: ванной, ванны, ванну; лемма «ванна» закроет остальные формы) |

**НЕ минусовать (осознанно):** черновой (планирование ремонта в новостройке — наша ЦА),
поэтапно/последовательность/пошагово (info-ЦА), однокомнатной/двухкомнатной (прикидка — идёт
в ключи), дизайн/идеи/варианты/фото/после (вдохновение), 2х/3х, проходной, дома, москве
(города НЕ минусуем в Этапе 2: прикидка гео-нейтральна; пересмотр по отчёту запросов).
**Уже правильно стоят:** под ключ, бригада, мастер (найм — вернуться после теста), капитальный
(ЖКХ), техника (холодильников/стиральных/машин…), кухни, туалет/ванная-семейство (→ Этап 3).

**Ключи ДОБАВИТЬ:** `"сколько стоит ремонт однокомнатной квартиры"` (6 287) и
`"сколько стоит ремонт двухкомнатной квартиры"` (3 696) — группа «Сколько стоит»;
`"варианты ремонта в хрущевке"` (1 681) — группа «Хрущевка».

## Этап 3 — Ванная и плитка (712722344, SUSPENDED; 43 минуса стоят)

**Главный мусор:** (а) реставрация САМОЙ ванны (эмаль/вкладыш/акрил — минусы уже стоят,
усилить); (б) техвыбор монтажных материалов («клей для плитки какой выбрать» 0.6k,
«гидроизоляцию» 0.3k); (в) города = найм местного подрядчика («ванная под ключ москва/спб/
пермь/казань» суммарно ~8k).
**Минусы ДОБАВИТЬ:**
| Слово | Почему |
|---|---|
| клей | монтажный техвыбор, не визуал |
| гидроизоляция, грунтовка | то же |
| наливной | наливной акрил = реставрация ванны |
| москва, спб, питер, петербург, казань, пермь | найм местного подрядчика (⚠️ спорное — см. ниже) |

**Спорное решение (владельцу):** города в Этапе 3. За минус: локальный запрос = ищут бригаду.
Против: часть — прикидка бюджета. Предлагаю: минусуем 6 городов из хвостов, остальное решит
отчёт «Поисковые запросы» после включения.
**НЕ минусовать:** программа/3д (ищут ИНСТРУМЕНТ раскладки — это мы), панелями (материал-
альтернатива), кабанчик/мрамор/белая/светлая (стили = ЦА), маленькой, душевой, хрущевке.

**Ключи ДОБАВИТЬ:** `"программа для раскладки плитки"` (~1.7k) и
`"варианты раскладки плитки в ванной"` (643) — группа «Плитка-визуал»;
`"плитка в маленькой ванной дизайн"` (561) — туда же.
