## Цель
Переоформить блок ссылки в `LinkAutofill` под конверсию (больше людей оставляют ссылку → реф-ссылка +
лид): рекламный заголовок с выгодами-галочками; убрать кнопку «Заполнить» → автоподгрузка параметров
при вводе ссылки (крутилка → поля). Плюс — зафиксировать РОАДМАП точного парсинга (per-site + ИИ-фолбэк).

## Источник задачи
Промпт владельца: сделать текст блока крупнее/рекламнее (галочки выгод); убрать кнопку «Заполнить» и
делать автоподгрузку при вводе ссылки (с индикатором загрузки); поля раскрывать и по «ввести вручную».
Плюс размышления о точности парсинга — просил мой план простым языком.

## Скоуп — что входит (этот деплой)
1. **Рекламный вид блока `LinkAutofill`:**
   - крупный заголовок «Добавьте ссылку на {accYour}» (жирнее/крупнее);
   - выгоды галочками (✓): «Заполним параметры расчёта сами», «Найдём, где дешевле, и пришлём вам»,
     «Подскажем, что ещё нужно купить»;
   - строка «Оставьте e-mail или подпишитесь на бот — сообщим, когда найдём выгоднее»; кнопки Телеграм/MAX/Почта — как есть.
2. **Убрать кнопку «Заполнить» → автоподгрузка:**
   - при вводе ссылки (debounce ~700 мс после остановки набора / на paste), если похоже на URL —
     авто-запрос `/api/calc/parse-link`; во время запроса — индикатор загрузки (крутилка/точки) под полем;
   - по ответу: авто-заполнить параметры + РАСКРЫТЬ поля (`MaterialParams`) ниже; при неудаче — раскрыть пустые поля с подсказкой «впишите вручную».
   - «ввести вручную» — так же раскрывает поля (без запроса).
3. **Парсер — без выдумывания:** заполняем ТОЛЬКО реально найденные поля (уже так); ничего не
   округляем/не домысливаем. (Косметика: проверить, что пустые значения не превращаются в 0.)

## Скоуп — что НЕ входит (роадмап парсинга — отдельные планы, по согласованию)
- **ИИ-фолбэк извлечения** (Gemini уже подключён): отдать текст страницы LLM, вернуть строго нужные
  поля, только найденное (JSON), ничего не выдумывать. Основной путь для незнакомых сайтов. — СЛЕДУЮЩИЙ шаг.
- **Per-site парсеры** под популярные магазины (Леруа/Максидом/Мир обоев/Стройпарк/ВсеИнструменты…):
  точные селекторы, растим по реальным ссылкам. — по мере поступления ссылок.
- Google-поиск НЕ используем (прямой fetch страницы точнее и надёжнее). Маркетплейсы с анти-ботом
  (Озон/ВБ) — честно ручной ввод (позже headless/фиды).
