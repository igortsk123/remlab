## Цель
ЗАХОД B большого батча `/calc/plitka` — пункты 5–6: убрать блок «Найдём выгоднее» и перестроить блок
ссылки (текст + Телеграм/MAX/почта-модалка + «ввести вручную»-аккордеон параметров + отдельной карточкой перед размерами).

## Источник задачи
Промпт владельца (6 пунктов) + ответы: боты БЕЗ «скоро»; блок ссылки — ОТДЕЛЬНОЙ карточкой (не внутри
секции), сначала ссылка+параметры, потом размеры. Заход A (п.1–4) уже выкачен.

## Ключевой контекст
- `FindCheaper` держал лид (email/боты/согласие); captureLead (`app/lead-actions.ts`) = email+urls+city+consent.
- Боты: `NEXT_PUBLIC_TELEGRAM_BOT`/`NEXT_PUBLIC_MAX_BOT` (env, могут быть пусты → кнопки-заглушки без «скоро»).
- `RoomPanel` сейчас ОДНА карточка со всем; `LinkAutofill`+`MaterialParams` внутри секции. `CalcBuilder`
  держит верхнюю кнопку «Если есть ссылка…» + `FindCheaper` (внизу) + `cheaperSignal` (липкую шапку из A — ОСТАВИТЬ).

## Скоуп — что входит
1. **Скрыть «Найдём выгоднее» (п.5):** убрать из `CalcBuilder` компонент `FindCheaper`, верхнюю кнопку
   «Если есть ссылка…» и `cheaperSignal`/`openSignal`. Липкую шапку итогов (A) оставить. Удалить `FindCheaper.tsx`.
2. **Блок ссылки (п.6) = отдельная карточка перед размерами.** `LinkAutofill` расширить в самодостаточный
   «материал-блок» (карточка):
   - текст: «Добавьте ссылку на выбранную вами {плитку…}. Мы заполним параметры сами. Если оставите
     e-mail или подпишетесь на бот, поищем, где дешевле и сообщим.» (по kind через `CALC_META.accYour`/`work`);
   - поле ссылки + зелёная «Заполнить» (как в A);
   - кнопки «Телеграм», «MAX», «Сообщить по почте». TG/MAX → deeplink (env; если пусто — кнопка-заглушка
     без перехода, без «скоро»). «Сообщить по почте» → `LeadModal`;
   - **`LeadModal`** (нов.): e-mail (проверка маски regex) + галочка «я согласен с политикой обработки ПДн»
     (ссылка-плейсхолдер), «Отправить» (по маске+согласию) → `captureLead({email, urls:[ссылка], kind, consent})`
     → закрыть; закрытие по клику на backdrop, по «×» справа, по Esc;
   - **«ввести вручную»** — невзрачная ссылка-кнопка; `MaterialParams` рендерится ВНУТРИ блока и по
     умолчанию СВЁРНУТ; раскрывается по «ввести вручную» ИЛИ автоматически после успешного автозаполнения/ввода ссылки.
3. **`RoomPanel` реструктуризация:** вместо одной карточки — стек карточек: [шапка комнаты] → для каждой
   плитки (стены; пол если добавлен): [карточка ссылка+параметры] → [карточка размеры] → [+ добавить размеры пола] → [итог].
   Порядок: ссылка/параметры ПЕРЕД размерами.
