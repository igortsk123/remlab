# _secrets — реестр доступов (ВНЕ git)

Здесь живут доступы проекта: сервисы, где лежат ключи, значения, как войти (`ACCESS.md`).

## Правила
- Папка ДОЛЖНА быть в `.gitignore` — apply-скрипт кита дописывает блок автоматически
  (`.memory_bank/_secrets/*` кроме этого README). **Проверь перед первым коммитом**;
  audit флагает `SECRET`, если папка есть, а игнора нет.
- В остальной памяти (`core/`, `decisions.md`, …) — только УКАЗАТЕЛИ
  («токен — см. `_secrets/ACCESS.md`»), никогда значения. Значение секрета в обычном
  memory-доке audit флагает как `SECRET`.
- Бэкап этой папки — забота владельца: git её сознательно НЕ хранит.
- Скиллы `/memory-check` и `/memory-cleanup` содержимое `_secrets/` не трогают.

## Формат ACCESS.md
Таблица: сервис · что (ключ/логин/URL) · значение или где лежит · как обновить/кто выдал.
