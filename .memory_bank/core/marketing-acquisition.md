## Спрос (Вордстат 2026-07-11, РФ, показов/мес)
- **AI-спрос ~15–18k** («ии/нейросеть дизайн интерьера», «по фото»; треть — «бесплатно» →
  оффер free) + **инструменты ~25–30k** («онлайн/программа/планировщик»).
- **«Дизайн проект квартиры» — 48k** («стоимость» ~15k) → под paid room pack.
- **Бренды конкурентов:** Remplanner ~35–40k, Planner 5D ~17k, Планоплан ~8k
  (Flatplan/MyNewRoom ≈0).
- **Широкие** («дизайн интерьера» 265k, «ремонт квартиры» 620k) — info/услуги → только РСЯ.
  ⚠️ 40–56% «комнатных» масок — ванная.
- **Неявный спрос:** «под ключ» 155k (аукцион подрядчиков — в лоб НЕ брать), мебель «недорого»
  ~150k (marketplace-интент, −50%/2г → только РСЯ-крео «примерь по фото»). Наш mid-funnel:
  «сколько стоит ремонт» 52k + «новостройка» 22k + «смета» ~6k ≈ 95k.
- **Сезонность:** ремонт июль–ноя (идёт); дизайн окт–янв. Гео: вся РФ, регионы дешевле.
