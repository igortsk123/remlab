# Market — Tier 1 (RU/UK, май 2026; v0.3)

## Контекст
В **UK**; продукт — под рынок РФ. Инфра (GPU/API) — UK-картой, без посредников; YooKassa/РФ-антифрод — только у конечных пользователей (Tier 2).

## Позиционирование
Побеждает **consumer renovation copilot**, а не «генератор картинок»: дефицит — связка **вкус → реальная комната → локальные товары → реализуемый проект**; moat — доведение от неуверенности до покупки и ремонта. Картинки/swipe — commodity; лидера полного цикла нет.

## 5 слоёв продукта
вкус (swipe) → захват комнаты (фото+размеры) → генерация+чат-правки → локальный подбор+смета → техпакет по обмерам. Игроки закрывают их поодиночке.

## Спрос
**RU:** DIY ~2,8 трлн ₽; хотят «товар+услугу»+консультанта; боли upsell/слабые теххарактеристики → прозрачные объяснения. **UK:** собственники, дорогие реновации, профи → confidence-before-spend.

## Оценки
RU **8/10** (ПДн 242-ФЗ); UK **7/10** (building regs); две страны ~**8/10** при «общий AI-core + локальные каталоги/compliance/execution» (Tier 2).

## Монетизация v0.3 (истина: `../../docs/master-brief-v0.3.md`)
Affiliate-first freemium: 1) **affiliate** (free, старт) — ~3% с выкупа (Гдеслон); 2) **paid room pack** — комната+Cost Engine+PDF, ~2 990 ₽ (цель по брифу; в коде — демо-пейволл 490 ₽, Cost Engine НЕТ); 3) **застройщики** (vision) — ~5% с чека. Гипотеза ступени 1: доход ≈ 2× CAC, kill — CAC > 2× дохода; цифры — Tier 2.

## Конкуренты
RU: Flatplan (ближайший), Remplanner, MyNewRoom, Planoplan. UK/глоб.: My Bespoke Room, IKEA Kreativ, John Lewis, Decor8 AI, Planner 5D / Homestyler.
