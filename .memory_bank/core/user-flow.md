# User Flow — Tier 1 сводка (freemium-граница — v0.3)

## Укороченный Stage 1 flow (7 экранов)
1. **Landing** — за 5с объяснить пользу; пример «фото→preview→товары(3 из 18)→бюджет→CTA». CTA: «Обновить комнату», «Рассчитать стоимость».
2. **Комната+цель+уровень** (один экран): гостиная/спальня; обновить/понять стоимость; уровень 1 освежить / 2 недорого обновить / 3 лёгкий косметический.
3. **Фото+бриф** (один экран): 1–3 фото, город, бюджет-диапазон, «что оставить», ограничения. Не спрашивать размеры/розетки до preview.
4. **Style cards**: фото + название стиля + акцент + объяснение; достаточно 3–5. → `style_profile`.
5. **AI-preview + подбор до N**: «AI понял комнату», 1 визуальный вариант (+дисклеймер «концепция, товары
   подобраны похожие»), 3–5 идей, **до N (3–5) РЕАЛЬНЫХ товаров из фидов с ценами и реф-ссылками, открыто**
   (+2–3 альтернативы), товарный бюджет (сумма цен).
6. **Paywall / room pack**: показать структуру платного, закрыть детали. После оплаты: ВСЕ элементы комнаты
   (8–20) + 3 альтернативы, цены/реф-ссылки, **Cost Engine** (стоимость работ), чек-лист, план, PDF, сохранение.
7. **Workspace «Мои комнаты»**: объект→комнаты→версии; обновить цены, заменить недоступные, PDF.

## Free vs Paid = уровень «что сделать с комнатой» (3 варианта, owner 2026-07-02)
1. **«Освежить без ремонта» — БЕСПЛАТНО:** AI-визуализация + подбор **до 3 реальных предметов мебели** по
   визуализации (открыто: фото/цена/реф-ссылка + 2–3 альтернативы, можно заменить) + товарный бюджет.
   Без материалов/сметы/плана.
2. **«Недорого обновить» — ПЛАТНО ~1 490 ₽:** мебель **в любом количестве** + подбор **материалов**.
   Без расчёта стоимости ремонта, без чертежей/дизайнера. Визуализация + реф-ссылки.
3. **«Ремонт под ключ» — ПЛАТНО 9 900 ₽:** мебель + материалы + гайды + инфо для строителей
   (**чертежи**) + **смета (Cost Engine)** + **услуга живого дизайнера**. Замена дизайнера, но в 5–10× дешевле.
- **Реф-ссылки во всех вариантах** (affiliate). **Paywall-триггер:** выбор варианта 2 или 3 (НЕ доступ к
  3 бесплатным товарам — они всегда открыты и кликабельны).
- ⚠️ **Код-долг:** структура «3 открыто, остальное закрыто» уже есть (`FREE_VISIBLE=3`, верна для вар.1);
  нужно: реальные товары из фидов Гдеслон + реф-ссылки (сейчас seed); добавить варианты 2/3 (материалы, любое
  кол-во, чертежи, Cost Engine, дизайнер); согласовать с `interventionLevel`.

## Второй вход: «найти похожую мебель по фото»
Короткий flow без генерации: до N элементов с фото (своё/найденное) → прямой поиск похожих в базе фидов
(pgvector). Самостоятельная точка входа, технически проще основного flow.

## Сегмент F — дизайнер интерьера (B2B, Stage 1.5)
Опт: подбор аналогов пачками. Нужны batch-режим (несколько комнат), выгрузка таблица/PDF, без брендинга в
клиентских материалах (white-label позже). Отдельный CJM. Цена ~990 ₽/комната от 10.

## Cost-сценарий (Stage 1B)
Известные данные → недостающее (площадь, потолок, состояние стен/пола, уровень материалов) → AI предлагает scope → пользователь подтверждает → **Cost Engine** считает диапазон + структура + confidence (low/mid/high). Сумму считает движок, НЕ LLM.

## Аналитика
PostHog события по всем экранам (landing_*, room/goal/intervention, photo/brief, style_card_*, ai_preview_*,
paywall_*, payment_*, workspace_*, cost_*) + **affiliate** (`affiliate_link_clicked`,
`affiliate_postback_received`, `affiliate_purchase_attributed`) + **лимиты генерации** (anti-abuse анонимов).
Полный список — Tier 2.
