## Цель
Полный **справочник размеров объектов жилых помещений** (средние + погрешность + ЖЁСТКИЕ границы правдоподобия),
в который движок замера смотрит для само-проверки — чтобы не было «тупых ошибок» (дверь 1 м, окно эркера 3 м).
Заменяет стартовую ручную `mltest/sizes_prior.py` (~30 классов) на подробный каталог по типам комнат.

## Источник задачи
Owner 2026-07-06: «найти готовый справочник или сгенерировать максимально подробный — сначала план какие объекты
бывают в жилых помещениях разного типа, пройтись по всем, проставить средние + погрешность; модель смотрит туда.
Чтобы дверь <180 = ошибка, стеклопакет эркера >2.5м = ошибка». Контроль room1: центр.окно ~100, боковые ~60.

## Готовые источники (сначала брать их, не выдумывать)
- **Neufert «Architects' Data»** — авторитет по размерам мебели/дверей/окон/зазоров.
- **dimensions.com** — структурированная база реальных размеров (по объектам).
- **Фиды Гдеслон** — реальные Ш×Г×В наших товаров (перцентили по категориям) — самый точный для мебели РФ.
- **ГОСТ РФ** — двери (полотно 2000×600-900), окна, потолки (2500-2700).
Схема: seed из Neufert/dimensions.com → уточнение перцентилями из фида → фикс-границы из ГОСТ.

## Схема записи (на каждый объект)
```
class: door
room_types: [any]
dims: { h:{avg:200, tol:5, min:180, max:230},   # min/max = ЖЁСТКИЕ границы (вне = ошибка)
        w:{avg:80,  tol:15, min:60,  max:120} }
measure: [h, w]            # какие измерения важны
source: gost | neufert | feed | dimensions.com
```
`tol` — обычная погрешность (жёлтая коррекция). `min/max` — жёсткие границы (красный флаг «ошибка замера»).

потолок, стена, пол, плинтус, карниз, радиатор/батарея, розетка/выключатель, штора/жалюзи, люстра/светильник,
зеркало, картина, ковёр, кондиционер.
**Гостиная:** диван (2-3-местный, угловой), кресло, пуф, журнальный стол, ТВ-тумба, телевизор, стеллаж,
книжный шкаф, комод, торшер, растение, тумба.
**Спальня:** кровать (одно/полутора/двуспальная), матрас, прикроватная тумба, шкаф (распашной/купе), комод,
туалетный столик, банкетка/пуф.
**Кухня:** гарнитур (нижние/верхние шкафы), столешница, холодильник, плита/варочная, духовка, вытяжка,
посудомойка, микроволновка, мойка, обеденный стол, стул, барный стул.
**Ванная/санузел:** ванна, душевая кабина, унитаз, раковина, тумба под раковину, стиральная машина,
полотенцесушитель, зеркало-шкаф, бойлер.
**Прихожая:** шкаф-купе, вешалка/панель, обувница, консоль, зеркало, пуф, ключница.
**Детская:** кровать детская, двухъярусная, письменный стол, стул, шкаф, стеллаж, комод-пеленальный, манеж.
**Кабинет/офис:** письменный/компьютерный стол, офисное кресло, стеллаж, шкаф для документов, тумба, монитор.
**Техника:** телевизор, монитор, ноутбук, вентилятор, обогреватель, увлажнитель/очиститель, колонка, роутер,
пылесос, гладильная доска.

## Жёсткие границы — примеры (детекторы «тупых ошибок»)
- Дверь-полотно: высота 180-230 (НИКОГДА <180); ширина 60-120.
- Межкомнатное окно/створка: высота 60-180; стеклопакет эркера длиной **≤250** (80% случаев).
- Подоконник от пола 70-100 (≥0 всегда).
- Потолок жилой 240-320.
- Кровать двуспальная ширина 140-200; матрас высота 15-35.
- Диван высота 75-100, глубина 80-105.

## Геометрические кросс-чеки (не из таблицы, из сцены)
- **Потолок-ограничитель:** верх окна/двери ≤ потолок; дверь ≤ потолок и ≥180; подоконник>0. (знаем потолок ~263 room1).
- Объект не выше потолка; footprint напольного лежит на полу; сумма стен ≈ периметр.
